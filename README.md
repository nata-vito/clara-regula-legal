# Clara Regula Legal - Assistente IA para Regularização Imobiliária

Este projeto é a interface e landing page para a **Clara**, uma Assistente de Inteligência Artificial focada em guiar profissionais (arquitetos, engenheiros, advogados e gestores) no processo de licenciamento e regularização de imóveis.

A aplicação foi construída utilizando:
- **React 19**
- **TanStack Start & Router**
- **Vite**
- **Tailwind CSS v4** + **Radix UI** (shadcn/ui)

## 🚀 Como rodar a aplicação localmente

### Pré-requisitos
- **Node.js**: Recomenda-se ter o Node.js na versão **18.x** ou **20.x** (LTS) instalada.
- **npm**: (Vem por padrão junto com o Node.js).

### Passo 1: Instalar as dependências

Abra o terminal na raiz do projeto e execute:

```bash
npm install
```

### Passo 2: Rodar o servidor de desenvolvimento

Para iniciar a aplicação em modo de desenvolvimento com Hot-Module Replacement (HMR) ativado, execute:

```bash
npm run dev
```

Após o script inicializar, o Vite disponibilizará um link no próprio console (normalmente `http://localhost:5173`). Acesse-o no seu navegador.

---

### Outros comandos úteis

- `npm run build`: Cria a build otimizada da aplicação (para produção).
- `npm run preview`: Serve localmente os arquivos gerados pela build para testes de produção.
- `npm run lint`: Executa a validação de código utilizando o ESLint para encontrar e corrigir erros.
